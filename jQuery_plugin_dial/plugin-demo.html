jQuery Plugin Demonstration For Rotating a Dial

This page demonstrates a user interface dial component implemented as jQuery plugin. It includes callback functionality, which can be seen in the “Event Log” section of this page.

"Dial" Control

This plugin demonstrates low-level mouse event handling. It allows the user to rotate the dial and they can then use it as a switch function. In my example it is used to switch the dot below from visible to invisible.
